# IA Insight

Site publico do AI Insights: hub premium em portugues para curadoria, analise e publicacao automatizada de conteudos sobre inteligencia artificial.

## Publicacao rapida

Este projeto e estatico. Pode ser publicado diretamente na Vercel, GitHub Pages, Netlify ou Cloudflare Pages.

Arquivo principal: `index.html`

## Automacao editorial

- `content.json`: conteudo publicado.
- `research-sources.json`: fontes de pesquisa.
- `editorial-policy.json`: criterios do gestor editorial.
- `scripts/`: automacoes de pesquisa, aprovacao e publicacao.
